# EMS Logger v2.0.2

## Что нового

- Исправлен общий cooldown, из-за которого разные действия могли не сохраняться в течение 15 секунд после предыдущего скриншота.
- Cooldown теперь считается отдельно для каждой папки/действия.
- Ошибки сохранения и захвата экрана теперь отображаются в статусе программы.

- Исправлено сохранение скриншотов в папки с кириллицей и Unicode-символами в пути.

## Основные возможности

- Полностью обновленный интерфейс.
- Выбор монитора и области OCR.
- Встроенный Tesseract OCR.
- Сортировка скриншотов по МСК.
- Настраиваемые фразы-триггеры и папки.
- Автоматическая проверка обновлений через GitHub Releases.
- Сохранение размера и положения окна.
- Дисклеймер при первом запуске.

## Установка

1. Скачайте `EMS_Logger_v2_0_2_Windows.zip`.
2. Распакуйте архив.
3. Запустите `EMS_Screen.exe`.
