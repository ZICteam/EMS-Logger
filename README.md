# EMS Logger v2.0.2

Темная Windows-утилита для EMS-логирования в Majestic RP: читает выбранную область экрана через OCR, находит заданные фразы-триггеры и сохраняет скриншоты по папкам с сортировкой по московскому времени.

> Дисклеймер: EMS Logger не связана и не поддерживается Majestic РП, Take-Two, Rockstar North Interactive или любым другим правообладателем. Все используемые товарные знаки принадлежат их соответствующим владельцам.

## Возможности

- Выбор монитора из реально подключенных экранов.
- Выбор области распознавания по снимку выбранного монитора.
- Встроенный Tesseract OCR в релизной сборке.
- Распознавание русского и английского текста.
- Сохранение скриншотов по фразам-триггерам и папкам.
- Сортировка скриншотов по МСК: `day` / `night`.
- Компактный интерфейс EMS Logger v2.0.
- Запоминание размера и позиции окна.
- Одноразовый дисклеймер при первом запуске.
- Проверка обновлений через GitHub Releases.
- Кнопка пожертвований: [DonationAlerts](https://www.donationalerts.com/r/zic_team).
- Поддержка кириллицы и Unicode-символов в пути к папке сохранения.

## Быстрый старт

1. Скачайте архив `EMS_Logger_v2_0_2_Windows.zip` из раздела Releases.
2. Распакуйте архив в удобную папку.
3. Запустите `EMS_Screen.exe`.
4. Выберите монитор для распознавания.
5. Нажмите `Выбрать область` и выделите область текста на превью монитора.
6. Проверьте триггеры во вкладке `Настройки`.
7. Нажмите `Запуск`.

## Настройка

Во вкладке `Настройки` можно изменить:

- фразы-триггеры;
- папки, куда сохраняются скриншоты;
- время начала дня и ночи;
- папку сохранения;
- проверить наличие обновлений.

Конфигурация хранится рядом с программой в config.json.

Путь к папке сохранения может содержать кириллицу, например `D:\Скриншоты\EMS`.

## Обновления

Приложение проверяет новую версию через GitHub Releases. Для корректной работы перед публикацией репозитория проверьте константу в `EMS_Screen.py`:

```python
GITHUB_REPOSITORY = "ZICteam/EMS-Logger"
```

Релизы должны иметь теги вида `v2.0`, `v2.1`, `v3.0.0`.

## Сборка из исходников

Требуется Windows и Python 3.12+.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m PyInstaller --clean EMS_Screen.spec
```

Готовый файл будет создан здесь:

```text
dist\EMS_Screen.exe
```

Для релизной сборки в корне проекта должна быть папка `tesseract` с `tesseract.exe` и языковыми файлами `rus.traineddata`, `eng.traineddata`, `osd.traineddata`.
