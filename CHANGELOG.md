# Changelog

## v2.0

- Новый компактный темный интерфейс.
- Выбор области распознавания.
- Выбор монитора из списка подключенных экранов.
- Чтение области через OCR.
- Встроенный Tesseract OCR в релизной сборке.
- Зеленая индикация корректно распознанного текста.
- Красная индикация ошибок OCR.
- Сортировка скриншотов по московскому времени.
- Запоминание размера и позиции окна.
- Одноразовый дисклеймер при первом запуске.
- Кнопка пожертвований.
- Проверка обновлений через GitHub Releases.
